UIElementsApp
=============

Descripción
-----------
Esta aplicación es una demostración práctica de los principales elementos de
interfaz de usuario (UI) en Android. Su objetivo es mostrar cómo funcionan
diferentes componentes gráficos mediante ejemplos simples e interactivos.

La aplicación cuenta con una Activity principal que contiene 5 Fragments,
cada uno dedicado a un grupo de elementos:

1. TextFields (EditText) → Entrada de texto como nombre y correo.
2. Botones (Button e ImageButton) → Acciones que muestran mensajes al usuario.
3. Elementos de selección (CheckBox, RadioButton, Switch) → Permiten elegir
   opciones o activar/desactivar configuraciones.
4. Lista (ListView) → Una lista de compras de 20 elementos de ejemplo.
5. Información (TextView, ImageView, ProgressBar) → Datos generales sobre
   la aplicación, el autor y ejemplos de presentación visual.

La navegación entre fragments se realiza con un menú inferior (BottomNavigationView).

------------------------------------------------------

Instrucciones de uso
--------------------
1. Al abrir la app, se cargará por defecto la pantalla de **TextFields**.
   Aquí puedes escribir tu nombre y correo en los campos de texto.

2. Usa la barra inferior de navegación para moverte entre las secciones:
   - **TextFields** → Introduce texto de ejemplo.
   - **Botones** → Prueba el botón normal y el botón con imagen.
     Al presionarlos, aparecerá un mensaje (Toast).
   - **Selección** → Marca y desmarca CheckBoxes, selecciona una opción de
     RadioButton, y prueba el Switch.
   - **Lista** → Revisa la lista de compras con 20 elementos.
   - **Info** → Consulta la descripción de la app y elementos visuales
     como texto, imágenes y una barra de progreso.

3. Explora cada fragmento para familiarizarte con el uso de los diferentes
   elementos UI. Todos los componentes tienen ejemplos sencillos para que
   entiendas rápidamente cómo se utilizan.

------------------------------------------------------

Requisitos
----------
- Android Studio (versión recomendada: 2021.3 o superior).
- SDK de Android configurado (mínimo API 21).
- Dispositivo o emulador con Android 5.0 o superior.

------------------------------------------------------

Notas
-----
- La app es únicamente de demostración académica, no almacena datos.
- Puedes modificar los textos, íconos y ejemplos de cada fragment según tus necesidades.
- El proyecto está organizado en fragments independientes para facilitar su comprensión y mantenimiento.
